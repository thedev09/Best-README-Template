## About The Project

[![Product Name Screen Shot][product-screenshot]](https://airsphere.tech)

Our project offers a view in the world of hosting and development of websites. We have shown how to host websites on AWS using AWSâ€™s Elastic Cloud Computing (EC2) services and storage on Simple Storage Service(S3) services. The  Development of our web-site was done using HTML,CSS and JavaScript for the front end. For backend we used Python and for DBMS MySQL.
